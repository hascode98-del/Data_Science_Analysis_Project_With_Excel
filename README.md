# Data Jobs Salary Analysis (Excel Project)

## Project Overview
This project analyzes a dataset of global data-related job postings using Microsoft Excel.  
The goal is to explore trends in **job roles, salaries, required skills, and job locations** in the data industry.

The project demonstrates practical **Excel skills used in data analysis**, including cleaning, organizing, and exploring data to generate insights.

## Dataset
The dataset contains information about job postings in the data field, including:

- Job Title
- Company Name
- Job Location
- Job Type (Remote / On-site)
- Average Yearly Salary
- Average Hourly Salary
- Required Skills
- Job Posting Platform


## Tools Used
- Microsoft Excel

## Excel Skills Demonstrated
- Data Cleaning
- Data Organization
- Table Formatting
- Basic Excel Formulas
- Data Exploration

## Project Objective
The objective of this project is to practice **data analysis using Excel** and understand trends in the **global data job market**, including salary distribution and skill demand.
